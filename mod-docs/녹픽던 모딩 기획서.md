# (중요)소수점 관련해서 특별히 정하지 않은 경우 전부 반올림 처리

> 근접 공격 기본 피해량

      - 최소 1+강화수치
      - 최대 2×티어+강화수치

> 탄환 기본 피해량

      - 최소 1×(티어+1)+강화수치
      - 최대 4×(티어+1)+강화수치×(티어+1)

> 탄환 기본 발사 속도

      - 1턴당 1발, 특수한 경우는 따로 적어 놓음

>### 1티어

* #### 조잡한 권총

      - 근접 피해 : 기본 피해
      - 탄환 피해 : 기본 피해

      - 특수 효과 없음

      - 장탄수 4발
      - 장전 시간 1턴

>### 2티어

* #### 자동권총

      - 근접 피해 : 기본 피해
      - 탄환 피해 : 기본 피해/2

      - 발사 속도 2배

      - 장탄수 6발
      - 장전 시간 1턴

* #### 나팔총

      - 근접 피해 : (티어 + 강화수치) - ((3 + 강화수치) * (티어+1))
      - 탄환 피해 : 기본 피해/2

      - 적이 1 - 5칸 떨어져 있을 경우 탄환 피해량 4 - 0배

      - 장탄수 1발
      - 장전 시간 3턴

* #### 리볼버

      - 근접 피해 : 기본 피해
      - 탄환 피해 : 기본 피해

      - 탄환 정확성 +30%

      - 장탄수 6발
      - 장전 시간 1턴



>### 3티어

      - 0 - (적과의 거리)만큼 데미지 추가, 최대 10, 강화 당 최대치 1씩 증가
      - 근접이 아닐 경우 탄환 정확성 +20%
      - 근접에서의 탄환 정확성 -70%

      - 장탄수 5발
      - 장전 시간 2턴


* #### 산탄총
       
      - 근접 피해 : (티어 + 강화수치) - ((3 + 강화수치) * (티어+1))
      - 탄환 피해 : (기본 피해/2)

      - 적이 1 - 4칸 떨어져 있을 경우 탄환 피해량 3 - 0배
      - 밀쳐내기 기능 제거

      - 장탄수 2발
      - 장전 시간 2턴

* #### 쌍권총

      - 근접 피해 : 기본 피해/2
      - 탄환 피해 : 기본 피해/2

      - 근접 공격 속도 2배
      - 발사 속도 2배

      - 장탄수 8발
      - 장전 시간 2턴

* #### 작살총

      - 근접 피해 : 기본 피해
      - 탄환 피해 : 기본 피해

      - 다트를 탄환으로 사용, 재장전 시 다트 선택, 퀵슬롯으로 재장전 시 일반 다트를 장전, 없을 경우 턴을 소모하지 않으며 재장전 실패(탄환이 없을 때의 메시지 출력)

      - 최대 장탄수 2발
      - 장전 시간 1턴

>### 4티어

>### 5티어

* #### 기관총

      - 근접 피해 : 기본 피해
      - 탄환 피해 : 기본 피해/2

      - 1번에 4발 발사
      - 탄환 정확성 -30%
      - 근접에서의 탄환 정확성 -20%
      
      - 장탄수 40발
      - 장전 시간 4턴

* #### 화염방사기

      - 근접 피해 : 기본 피해
      - 탄환 피해 : 기본 피해/2

      - 전방 3칸에 불을 뿜고 탄환 피해만큼의 피해를 줌, 불이 덮인 타일을 1턴 동안 불태움
      - 연속해서 발사할 시 피해량 +50% 적용
      - 재장전에 액체 화염 물약 사용
      - 액체 화염 물약이 없을 경우 액체 금속으로 장전 가능, 3개 소모
      
      - 장탄수 10발, 표기는 10 * (장탄수)%/100%로 표기
      - 장전 시간 2턴

* #### 플라즈마 포

      - 근접 피해 : 기본 피해
      - 탄환 피해 : 기본 피해

      - 발사 속도 0.5배
      - 발사 시 발사 방향의 반대 방향으로 1칸 이동
      - 착탄지점 중심에 탄환 피해의 100%, 반경 1칸에 67%의 피해를 주고, 3x3지역에 전격 폭풍을 2턴간 생성
      - 재장전에 액체 금속 사용
      - 유물 충전 효과가 있는 경우 (주문서로 얻는 유물 충전 효과를 전부 받을 경우)총 8의 장탄수 회복
      - 1턴마다 1/25(1%)의 장탄수 회복
      
      - 장탄수 4발, 표기는 25 * (장탄수)%/100%로 표기
      - 장전 시간 3턴
